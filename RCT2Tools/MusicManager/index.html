Features
Author: Robert Jordan
Version: v1.1.0.1
Rollercoaster Tycoon 2 supports up to two custom songs for rides, RCT2 Music Manager takes advantage of that and creates a simple system for switching out which songs to use. This program will also convert any songs in the MP3, M4A, AIFF, or WAVE format into the correct format for use with RCT2. RCT2 does support changes to the custom songs in game and should update almost instantly. However if the new song is longer than the previous song, it will only last as long as that song until the game is restarted.

Controls
Play/Stop
Play the currently selected song or stop the currently playing song.
Set Custom Song
Set the custom song to the currently selected song.
Volume
Change the volume of the currently selected song. The volume is always relative to the original volume of the song. Songs with modified volume will have a '*' in the V column.
Rename
Rename the currently selected song.
Delete
Deletes the currently selected songs.
Browse Data Directory
Change the directory to RCT2's Data directory where it stores custom songs.
Refresh
Refresh the song list.
Delete Key
Deletes the currently selected songs.
Double Click Song
Will play or restart the selected song.
Drag Files Onto List
Allows you to add more songs to the collection. Any song that doesn't meet the required format will be converted.
