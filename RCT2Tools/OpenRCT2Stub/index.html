Instructions
  • Find your Steam installation folder. It should be in Program Files (x86)
  • Goto Steamapps/Common/RollerCoaster Tycoon 2
  • Rename "RCT2" to "Vanilla"
  • Place the contents of this zip file into your Steam RCT2 directory
  • Download the latest version of the OpenRCT2 Launcher from OpenRCT.net
  • Place the contents of the downloaded launcher zip file into your Steam RCT2 directory
  • Make sure the OpenRCT2 Launcher is named "OpenRCT2 Launcher"
  • You can now launch the stub through Steam, but there's one more step
  • In order for OpenRCT2 to work with the stub, you'll have to update OpenRCT2 from the launcher at least once. Otherwise the stub won't be able to find OpenRCT2.
  • If OpenRCT2 isn't found or crashes on startup try updating from the launcher again.
  • Now you should be able to launch OpenRCT2 from the stub. If you want the Steam overlay to work then enable "Hardware display" in the in-game options menu. Note: Hardware display only works when your DPI is at 100%
