Metroid Map Guardian

This is a program I created to simplify collecting power-ups in the Metroid Prime series. Many thanks to Falcon Zero for letting me use his maps and guides he created for the 3 games.

Ascii Artist

A program for creating and saving images drawn in the console using ASCII characters and colors. This program has gone through numerous versions. I often use it as a grid-based level editor.

Requirements: Windows 7
Download Link | Image Album

AudioVis

AudioVis is a program created in flash that allows you to enjoy your music through customizable visualizations.

Requirements: Flash Player, Java 1.6 (or higher, for creating playlists)

Julia Set Explorer

A program for creating and viewing Julia Set fractals. You can choose different functions and customize the colors.

Download Link | Image Album

RCT2Tools

A collection of programs (currently only one) for working with Rollercoaster Tycoon 2. The current program is called RCT2 Content Browser. With it you can view all objects in your RCT2 directory making it easier to manage custom content. Detailed information is also given about the file format.

Requirements: Windows
